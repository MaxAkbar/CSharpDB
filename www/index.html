Why CSharpDB?

A complete embedded database platform — from storage primitives to SQL to ETL.

Zero Dependencies

No external NuGet packages. Pure .NET implementation from page management to query execution.

ACID Transactions

Full atomicity, consistency, isolation, and durability. WAL-based recovery with snapshot isolation.

Full SQL Engine

DDL, DML, JOINs, aggregates, CTEs, subqueries, views, triggers, and stored procedures.

Collection API

Typed Collection<T> with JSON serialization, nested path indexing, and range queries.

B+Tree Storage

Efficient on-disk key-value storage with automatic node splitting, cursor iteration, and range scans.

ETL Pipelines

Built-in pipeline runtime with CSV/JSON sources, transforms, and destinations. Visual designer in Admin UI.

Snapshot Isolation

Concurrent readers never block the writer. Each reader sees a consistent point-in-time snapshot.

Multiple Access Layers

ADO.NET provider, REST API, gRPC daemon, CLI REPL, Client SDK, MCP server, and VS Code extension.

Single-File Database

Your entire database in one file. 4096-byte slotted pages with efficient freelist management.

Three Ways to Work

Use SQL, the Collection API, or low-level storage — all in one engine.

SQL via ADO.NET 
using var conn = new CSharpDbConnection("Data Source=myapp.db");
await conn.OpenAsync();

await using var cmd = conn.CreateCommand();
cmd.CommandText = @"
    CREATE TABLE IF NOT EXISTS Users (
        Id   INTEGER PRIMARY KEY,
        Name TEXT NOT NULL,
        Email TEXT
    )";
await cmd.ExecuteNonQueryAsync();

cmd.CommandText = "INSERT INTO Users VALUES (@id, @name, @email)";
cmd.Parameters.Add(new("@id", 1));
cmd.Parameters.Add(new("@name", "Alice"));
cmd.Parameters.Add(new("@email", "alice@example.com"));
await cmd.ExecuteNonQueryAsync();
Collection API 
await using var db = await Database.OpenAsync("myapp.db");
var users = await db.GetCollectionAsync<User>();

// Put, Get, Delete
await users.PutAsync(1, new User("Alice", "alice@example.com"));
var alice = await users.GetAsync(1);

// Index and query
await users.Index.OnAsync(u => u.Name);
var results = await users.FindAsync(u => u.Name, "Alice");
