Getting Started

Get CSharpDB up and running in your .NET project in minutes. Choose the API that fits your needs.

Installation

dotnet add package CSharpDB

This meta-package includes the storage engine, SQL engine, Collection API, ADO.NET provider, and diagnostics. For finer control, install individual packages like CSharpDB.Storage, CSharpDB.Engine, or CSharpDB.Data.

Option 1: SQL via ADO.NET

The most familiar approach for .NET developers. Use standard DbConnection, DbCommand, and DbDataReader APIs.

using CSharpDB.Data;

using var conn = new CSharpDbConnection("Data Source=myapp.db");
await conn.OpenAsync();

// Create a table
await using var cmd = conn.CreateCommand();
cmd.CommandText = @"
    CREATE TABLE Products (
        Id    INTEGER PRIMARY KEY,
        Name  TEXT NOT NULL,
        Price REAL
    )";
await cmd.ExecuteNonQueryAsync();

// Insert rows with parameters
cmd.CommandText = "INSERT INTO Products VALUES (@id, @name, @price)";
cmd.Parameters.Clear();
cmd.Parameters.Add(new("@id", 1));
cmd.Parameters.Add(new("@name", "Widget"));
cmd.Parameters.Add(new("@price", 9.99));
await cmd.ExecuteNonQueryAsync();

// Query
cmd.CommandText = "SELECT Id, Name, Price FROM Products WHERE Price > 5.0";
await using var reader = await cmd.ExecuteReaderAsync();
while (await reader.ReadAsync())
{
    Console.WriteLine($"{reader.GetString(1)}: ${reader.GetDouble(2):F2}");
}

Option 2: Engine API with SQL

Use the Database class directly for a lightweight, high-performance approach.

using CSharpDB.Engine;

await using var db = await Database.OpenAsync("myapp.db");

// Execute SQL directly
await db.ExecuteAsync(@"
    CREATE TABLE Users (
        Id    INTEGER PRIMARY KEY,
        Name  TEXT NOT NULL,
        Email TEXT
    )");

await db.ExecuteAsync("INSERT INTO Users VALUES (1, 'Alice', 'alice@example.com')");
await db.ExecuteAsync("INSERT INTO Users VALUES (2, 'Bob', 'bob@example.com')");

var results = await db.ExecuteAsync("SELECT * FROM Users ORDER BY Name");
foreach (var row in results.Rows)
    Console.WriteLine($"{row[1]}: {row[2]}");

Option 3: Collection API (NoSQL)

Work with typed C# objects — no SQL needed. Built-in JSON serialization and indexing.

using CSharpDB.Engine;

public record Product(string Name, double Price, string[] Tags);

await using var db = await Database.OpenAsync("myapp.db");
var products = await db.GetCollectionAsync<Product>();

// Store objects
await products.PutAsync(1, new("Widget", 9.99, ["tools", "hardware"]));
await products.PutAsync(2, new("Gadget", 24.99, ["electronics"]));

// Retrieve
var widget = await products.GetAsync(1);

// Index a property and query
await products.Index.OnAsync(p => p.Name);
var results = await products.FindAsync(p => p.Name, "Gadget");

// Scan all
await foreach (var (key, product) in products.ScanAsync())
    Console.WriteLine($"{product.Name}: ${product.Price:F2}");

Option 4: Low-Level Storage API

For maximum control, use the storage engine directly with B+trees, pages, and the WAL.

using CSharpDB.Storage;

await using var ctx = await DefaultStorageEngineFactory
    .OpenAsync("myapp.db");

// Define schema
var schema = new TableSchema("Events", [
    new("Id",      DbType.Integer, isPrimaryKey: true),
    new("Name",    DbType.Text),
    new("Payload", DbType.Blob, nullable: true),
]);
await ctx.Catalog.CreateTableAsync(schema);

// Write transaction
await ctx.Pager.BeginWriteTransactionAsync();
var tree = ctx.Catalog.GetTableTree("Events");
await tree.InsertAsync(1, encoder.Encode([
    DbValue.FromInteger(1),
    DbValue.FromText("startup"),
    DbValue.Null,
]));
await ctx.Pager.CommitTransactionAsync();

Custom Configuration

Fine-tune the storage engine for your workload.

var options = new StorageEngineOptionsBuilder()
    .WithCacheSize(2048)
    .WithLockTimeout(TimeSpan.FromSeconds(10))
    .WithCheckpointPolicy(new FrameCountCheckpointPolicy(500))
    .Build();

await using var db = await Database.OpenAsync("myapp.db", options);

// Also available: in-memory and hybrid modes
await using var mem = await Database.OpenInMemoryAsync();
await using var hybrid = await Database.OpenHybridAsync("myapp.db");
