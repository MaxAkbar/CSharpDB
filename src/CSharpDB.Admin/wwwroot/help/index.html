CSharpDB Studio Help

CSharpDB Studio is the desktop administration workspace for browsing databases, running SQL, editing rows, designing schemas, building forms and reports, inspecting storage, and managing operational workflows.

Get started Forms and reports Keyboard shortcuts

The welcome screen shows Object Explorer, open tabs, connection state, and the main command surface.

What You Can Do

Browse and edit data

Open tables, views, and collections from Object Explorer. Filter, sort, page through rows, edit values, and save changes from a dedicated tab.

Run SQL

Use the query editor for SQL, formatting, saved queries, execution plans, results grids, and quick diagnostics.

Design schemas

Create and inspect tables, columns, indexes, triggers, views, procedures, and relationships without leaving the admin workspace.

Build forms and reports

Create Access-style data entry forms and printable banded reports with visual designers, previews, validation, and event hooks.

Move data

Import and export table archives, register external tables, and use the pipeline designer for repeatable ETL flows.

Inspect storage

Review database header details, WAL state, page histograms, free space, index health, and maintenance actions from Storage Inspector.

Help Bundle Scope

This help site is intentionally separate from the public CSharpDB web site. It contains admin-only guidance and local screenshots so it can be packaged with the CSharpDB Studio MSIX and opened without internet access.

MSIX packaging: these files live under src/CSharpDB.Admin/wwwroot/help. The admin web host publishes them as static assets, and the desktop Store package includes that publish output.
