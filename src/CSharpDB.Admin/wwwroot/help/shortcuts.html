Keyboard Shortcuts and Daily Workflow

Use shortcuts for navigation, query execution, layout changes, and tab management.

Keyboard Shortcuts

ShortcutAction
Ctrl+KOpen the command palette.
Ctrl+NOpen a new query tab.
Ctrl+BToggle Object Explorer.
Ctrl+EnterRun the current query from the SQL editor.
Ctrl+Shift+LToggle dark or light theme.
Ctrl+WClose the active closable tab.
Double-click cellEdit a cell in editable data grids.

Common Workflows

Find an object quickly

  1. Press Ctrl+K.
  2. Type part of the object name.
  3. Choose the table, view, form, report, procedure, or callback entry.

Inspect a table

  1. Open the table from Object Explorer.
  2. Use filters or sort headers to narrow records.
  3. Open the schema view for columns, indexes, and triggers.

Create a data entry screen

  1. Select a table and choose Design Form.
  2. Place controls on the canvas.
  3. Bind fields and configure validation.
  4. Open Data Entry to test the runtime form.

Build a report

  1. Create a report from the Reports group or a table context menu.
  2. Select a source and generate or place controls.
  3. Add groups, sorts, and calculated text where needed.
  4. Preview and print.

Troubleshooting

SymptomCheck

Callback is missingOpen callback readiness or diagnostics and confirm the host registered the expected function, command, or validation rule.
